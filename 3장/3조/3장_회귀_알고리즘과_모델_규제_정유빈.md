## 결정계수(R^2)
- R^2 = 1 - (타깃 - 예측)^2 의 합 / (타깃 - 평균)^2 의 합
- 1 - SSE/SST = SSR
- SST = SSR(설명이 가능) + SSE(설명X)
- 즉, SSR/SSR = R^2 의 값이 1에 가까울 수록 이 모델이 모두 설명할 수 있다고 해석된다.
- 평균 정도를 예측하는 수준이라면, R^2은 0에 가까워지고, 예측이 타깃에 가까워지면 1에 가까워진다.
